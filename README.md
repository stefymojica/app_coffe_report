# Asistente de Chat con LLMs para Análisis Histórico y Predictivo del consumo de Café

## Instalación
### Clonar repositorio 

````bash
git clone git@github.com:stefymojica/app_coffe_report.git
cd app_coffe_report
````

## Instalar dependencias

```bash
npm install
````

## Ejecutar App en modo desarrollo

```bash
npm run dev
````
